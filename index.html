무한매수법(Rao V4) 규칙 요약

1회매수금 = 현금 ÷ (N − T). 별% = max(0, base − slope×T). 별지점 = 평단 ×(1+별%). 별 LOC 매수가 = 별지점 − 0.01.

전반전(T < N/2): 별쪽 예산 = 1회매수금/2, 평단쪽 예산 = 1회매수금/2. 후반전(T ≥ N/2): 별쪽 예산 = 1회매수금 전액, 평단쪽 = 0.

쿼터 매도 = 보유수량/4(내림), 별지점에 LOC. 지정가 매도 = 평단 ×(1+finalsell), 나머지 전량. T 갱신: 매수 +1(전량)/+0.5(일부), 쿼터매도 ×0.75, 지정가매도 ×0.25.

분할수(N) 변형 프리셋의 slope는 엑셀 패턴(별% 이 T=N/2에서 0이 됨 → slope = 2×base÷N)에서 도출했습니다. 언제든 직접 수정 가능합니다.

VR(밸류리밸런싱) 규칙 요약

새 V = 직전 V + 직전 Pool/G + 적립 − 인출. 하단 = V×(1−밴드), 상단 = V×(1+밴드).

매수필요 = max(0, 하단−평가금), 매도필요 = max(0, 평가금−상단). Pool 한도 = (Pool+적립−인출)×한도%. 매수주수 = 내림(최대매수 ÷ (가격+수수료)).
